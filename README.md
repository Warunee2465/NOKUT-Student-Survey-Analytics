# NOKUT Student Survey – Tableau Dashboard

This repository contains a **Tableau dashboard** analysing the NOKUT Student Survey dataset.

### Purpose
Explore patterns in:
- student satisfaction
- learning environment
- programme quality

to understand where improvement opportunities exist in higher education programmes.

---

## Repository Contents

| File | Description |
|------|-------------|
| `NOKUT_Student_Survey.twbx` | Tableau packaged workbook (contains dashboard + data) |
| `screenshots/` | Preview images of the dashboard pages |
| `README.md` | Project documentation |

---

## Key Insights (example)

- satisfaction varies across faculties / fields of study
- learning environment scores highlight where improvements can be focused
- workload differences are linked to variation in satisfaction levels
