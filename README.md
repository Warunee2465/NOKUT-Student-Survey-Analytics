# NOKUT Student Survey – Data Analytics Dashboard

Tableau dashboard analysing NOKUT Student Survey results — visual insights on student satisfaction, learning environment and study quality across faculties/programmes.

**Goal:** Identify key patterns in student satisfaction, learning environment, and study quality in Norwegian higher education.

### What is inside

| File | Description |
|------|-------------|
| `NOKUT_Dashboard.pbix` | Power BI Dashboard file |
| `/data` (optional if included) | Supporting dataset files |
| `/screenshots` | Preview images of dashboard pages |

### Key Insights
- overall student satisfaction varies significantly between faculties
- learning environment scores show clear improvement opportunities
- programme workload differences strongly drive sentiment

### Tooling
- **Power BI**
- basic data cleaning using Power BI Power Query

### How to view
1. download the `.pbix` file
2. open with Microsoft Power BI Desktop

---

If you want the same dashboard in Tableau version, `.twbx` is included as well.
